Mirawatt - for TED 5000

Intructions: to add this an iPhone web app,

  • From your iphone, start the web browser (Safari)
  • Navigate to this page http://ted.mirawatt.com/
  • Add the web page as an app using the + button at the bottom of the Safari screen
  • This will install the app on your Home Screen with this icon:
  • Enjoy the app.
  • A swipe gesture on the graph (left of right) will change the view from Hour, Month, to Year etc...

Note: The simulation on the right has been tested on Firefox,Chrome and Safari. It does not work with Internet Explorer.
